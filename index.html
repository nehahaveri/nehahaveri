GigKart

Hyperlocal micro-gig marketplace · production-grade

A full marketplace where posters list local tasks (cleaning, tutoring, delivery) and taskers bid & complete them — payments held in Razorpay escrow, auto-released after 48 hrs, with disputes flowing to admins.

  • Geo-radius search using PostGIS ST_DWithin
  • Razorpay escrow + UPI payouts via Razorpay Route
  • Cron auto-release after 48 hrs of inactivity
  • Real-time chat & notifications via SSE
  • Fully Dockerized · JSON-LD SEO per listing

JustGP-Rx

End-to-end Healthcare Prescription Platform · Spring Boot Microservices

A fully deployed, production-grade healthcare SaaS — 8 decoupled Spring Boot services behind an API Gateway, JWT-secured auth, real-time prescription generation, and PDF uploads to AWS S3. Orchestrated with Docker Compose and shipped with CI/CD via GitHub Actions.

  • API Gateway routing to 8 independent services
  • JWT auth service — role-based access for doctors, receptionists
  • Prescription engine — instant PDF generation & storage on AWS S3
  • Fully Dockerized · multi-service Docker Compose orchestration
  • GitHub Actions CI/CD · zero-downtime deployments on cloud
